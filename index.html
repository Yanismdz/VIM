Basics

Move the cursor
Left -> h
Right -> i
Bottom -> j
Top -> k
Leave without saving
Escape:q! Enter
Save and leave
Escape:wq Enter
Delete a letter in normal mode
x
Turn into insert mode
i
Leave insert mode
Escape

Delete/Modify

Delete from the cursor to the end of the word
dw
Delete from the cursor to the end of the line
d$
Delete all the line
dd
Undo last action
u
Undo change on a line
U

Insertion

Open a line on insert mode under the cursor
o
Open a line on insert mode on top of the cursor
O
Insert text after the cursor
a
Insert text at the end of the line
A
Replace the letter on the cursor
r
Turn into replace mode
R
1

Cut/Copy/Paste

Cut the text
dd
Paste the text after the common position
p
Paste the text before or on top of common position
P

Research/Position

Display the position en stat of the file
Ctrl+G
Go to the end of the file
G
Research ahead
/
Research behind
?

External Command

Execute a commande
:!command
Display a directorie
:!ls
Remove a file
:!rm **name of the file**
Save the file
:w **name of the file**
